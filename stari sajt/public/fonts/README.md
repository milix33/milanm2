# Fonts Directory

Stavite vaše custom font fajlove ovde.

## Format fajlova:
- `YourFont-Regular.woff2` - Regular (400)
- `YourFont-Bold.woff2` - Bold (700)

Ili drugi formati:
- `.woff`
- `.ttf`
- `.otf`

## Primer strukture:
```
public/fonts/
  ├── YourFont-Regular.woff2
  ├── YourFont-Bold.woff2
  └── README.md
```

Fontovi će biti automatski optimizovani od strane Next.js.
